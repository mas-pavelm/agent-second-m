# Правила классификации записей

## Язык
Пользователь говорит по-русски. Все записи, теги и заметки — на русском.

## Типы записей

### ЗАДАЧА
Признаки: «нужно», «сделать», «позвонить», «написать», «проверить», глагол с дедлайном.
Действие: создать задачу в Todoist.

Приоритеты:
- P1 — деньги, договоры, объекты в работе, здоровье, критичные дедлайны
- P2 — найм (ГИ, B2B-менеджер, операторы EDM/VPG/LMD), НИОКР-контакты, встречи с партнёром, сын
- P3 — административное, «изучить/посмотреть»
- P4 — идеи без срока

Проекты Todoist:
- Производство металлообработка → Продажи, Оборудование, НИОКР VPG FL-HYDRO, НИОКР ИЛИСТ XL, Персонал
- Строительная компания → Клиенты и объекты, Главный инженер
- Личные финансы → Кредит, Инвестиционный счёт
- Здоровье
- Сын — университет
- Ежемесячный контроль

### ИДЕЯ
Признаки: «а что если», «можно попробовать», размышление без конкретного действия.
Действие: vault/thoughts/ideas/
Теги: #идея + #производство / #строительство / #НИОКР / #личное

### CRM / КЛИЕНТ / ПАРТНЁР
Признаки: имя + контекст встречи/звонка/договора.
Действие: vault/business/crm/ или vault/business/network/
Фиксировать: имя, компания, суть, следующий шаг, дата.

Важные контакты для распознавания:
- VPG — поставщик гидролазерного комплекса FL-HYDRO
- ИЛИСТ / СПбГМТУ — поставщик LMD-установки ИЛИСТ XL (ilwt@ilwt.smtu.ru)
- Партнёр / соучредитель — со-владелец производства

### ПРОГРЕСС ПО ЦЕЛЯМ
Признаки: выручка, вес, бассейн, кредит, этапы стратегии.
Действие: обновить vault/goals/ + запись в vault/daily/

### ЕЖЕМЕСЯЧНЫЙ КОНТРОЛЬ
KPI для фиксации каждый месяц:
- Выручка производства (факт vs план)
- Загрузка лазера (цель >70%)
- Рамочные договоры (цель к Q4 2026: 3+)
- Выручка строительства (цель 5–6 млн/мес)
- Остаток кредита + досрочный платёж
- Вес (еженедельно по пятницам)
- Бассейн (цель: 12–13 раз/мес)
- Статус найма: ГИ, B2B-менеджер
- Статус НИОКР: VPG FL-HYDRO, ИЛИСТ XL

### ОБУЧЕНИЕ / ИНСАЙТ
Действие: vault/thoughts/learnings/

### РЕФЛЕКСИЯ
Действие: vault/thoughts/reflections/

## Всегда P1
- Объекты в работе: проблемы, задержки, деньги
- Финансовые обязательства с дедлайном
- Договоры с дедлайном подписания

## Специальные категории
- Производство: лазер, листогиб, клиенты, B2B-менеджер, загрузка
- НИОКР: VPG FL-HYDRO (2027 H2), ИЛИСТ XL (2028 H2), субсидии ФРП, EDM
- Строительство: объекты, клиенты, вакансия ГИ
- Финансы: кредит ~4 млн, накопления, ОФЗ, дивиденды
- Сын: ЕГЭ, репетиторы, вузы, переезд в Москву
- Здоровье: бассейн, вес, медосмотр
- Семья: дочь (ипотека), супруга, отпуск
