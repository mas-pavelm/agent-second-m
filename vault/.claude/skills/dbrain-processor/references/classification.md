# Правила классификации записей

## Язык

Пользователь говорит по-русски. Все записи, теги и заметки — на русском.

---

## Типы записей и куда сохранять

### 📋 ЗАДАЧА
**Признаки:** «нужно», «сделать», «позвонить», «встретиться», «написать», «проверить», «заказать», глагол в инфинитиве с дедлайном или без.

**Действие:** создать задачу в Todoist.

**Приоритеты Todoist:**
- **P1 (срочно+важно):** деньги, договоры, объекты в работе, здоровье, критичные дедлайны
- **P2 (важно, не горит):** найм, партнёрства, стратегические встречи, сын (вуз, репетиторы)
- **P3 (может подождать):** административное, исследование, «посмотреть/изучить»
- **P4 (когда-нибудь):** идеи без срока, «было бы хорошо»

**Проекты Todoist:**
- `Строительство` — всё по строительной компании
- `Производство` — металлоконструкции, НИОКР, партнёр
- `Финансы` — кредит, накопления, инвестиции
- `Сын` — вуз, репетиторы, переезд
- `Здоровье` — бассейн, врачи, вес
- `Семья` — дочь, супруга, отпуск
- `Личное` — всё остальное

---

### 💡 ИДЕЯ
**Признаки:** «а что если», «можно попробовать», «идея», «мысль», размышление без конкретного действия.

**Действие:** сохранить в `vault/thoughts/ideas/`

**Теги:** #идея + тематический тег (#строительство / #производство / #личное / #инвестиции)

---

### 📞 CRM / КЛИЕНТ / ПАРТНЁР
**Признаки:** имя + контекст встречи/звонка/договора, упоминание клиента, заказчика, поставщика, партнёра.

**Действие:** создать или обновить карточку в `vault/business/crm/` или `vault/business/network/`

**Обязательно фиксировать:** имя, компания, суть разговора, следующий шаг, дата.

---

### 🎯 ПРОГРЕСС ПО ЦЕЛЯМ
**Признаки:** упоминание веса, дохода, выручки, бассейна (км/частота), погашения кредита, этапов стратегии.

**Действие:** обновить соответствующий файл в `vault/goals/` + запись в `vault/daily/`

---

### 📚 ОБУЧЕНИЕ / ИНСАЙТ
**Признаки:** «понял», «узнал», «прочитал», «вывод», урок из ситуации.

**Действие:** сохранить в `vault/thoughts/learnings/`

---

### 🔄 РЕФЛЕКСИЯ
**Признаки:** эмоциональная оценка дня/ситуации, «устал», «доволен», «беспокоит», личные мысли.

**Действие:** сохранить в `vault/thoughts/reflections/`

---

### 📸 ФОТО / ДОКУМЕНТ
**Признаки:** изображение, скан, фото объекта/доски/документа.

**Действие:** описать содержимое, сохранить в `vault/daily/` с тегом #фото. Если это чертёж/смета — в `vault/business/`.

---

## Приоритизация задач — правило для агента

### Всегда срочно (P1):
- Действующие объекты: проблемы, задержки, деньги
- Здоровье: врачи, анализы (если назначено)
- Финансовые обязательства с дедлайном (платёж по кредиту и т.д.)
- Договоры с дедлайном подписания

### Важно, не горит (P2):
- Найм главного инженера (стратегический приоритет)
- Встреча с партнёром по производству
- Репетиторы / вуз для сына
- Бронирование отпуска

### Можно отложить (P3–P4):
- Изучить / посмотреть / почитать
- Административное без срока
- Идеи без плана

## Специальные категории (важны для этого пользователя)

| Категория | Что включает |
|-----------|-------------|
| **Строительство** | Объекты, клиенты-заказчики, бригады, сметы, вакансия ГИ |
| **Производство** | Металлоконструкции, НИОКР, партнёр-соучредитель, EDM-станок |
| **Финансы** | Кредит (~4 млн), накопления, инвестиции (ОФЗ), дивиденды |
| **Сын** | Вуз, ЕГЭ, репетиторы, переезд в Москву, поездки на дни открытых дверей |
| **Здоровье** | Бассейн (частота + дистанция), вес (еженедельно), медосмотр |
| **Семья** | Дочь (ипотека), супруга, отпуск (планирование) |
| **Горы** | Трекинг, маршруты, снаряжение — соло или с сыном |

---

## Что НЕ делать

- Не создавать задачи из рефлексий — только сохранять
- Не дублировать идеи как задачи (если нет конкретного действия)
- Не ставить P1 на «изучить / посмотреть / подумать»
- Не теряться в деталях — важна суть и следующий шаг
