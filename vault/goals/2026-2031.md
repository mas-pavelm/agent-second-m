# Цели по годам 2026–2031

- [ ] Выбрать нишу: АПК + промышленное оборудование
- [ ] Нанять менеджера B2B-продаж (апрель–май)
- [ ] Аудит загрузки оборудования (апрель)
- [ ] Обзвон 20 бывших клиентов (апрель)
- [ ] 3+ рамочных договора с промпредприятиями (к Q4)
- [ ] Лазер 6–8 кВт: КП (май), договор (Q3), запуск (Q4)
- [ ] Выручка: 30 → 50 млн/год
- [ ] Первый контакт с VPG FL-HYDRO (май)
- [ ] Первый контакт с ИЛИСТ СПбГМТУ (май)

Строительство:
- [ ] Аудит воронки клиентов (апрель)
- [ ] Профиль ГИ + вакансия (Q3–Q4)
- [ ] Выручка: 60–70 млн/год

- [ ] График погашения кредита составлен (апрель)
- [ ] Погашено 2,5–3 млн от кредита

Личное:
- [ ] Бассейн: 3 раза/нед к Q3
- [ ] Медосмотр (апрель/май)
- [ ] Вес: −4–6 кг (→ 93–95 кг)
- [ ] Горный однодневный маршрут
- [ ] Отпуск забронирован (май)
- [ ] Разговор с сыном о вузе, репетиторы найдены

- [ ] Покрасочная камера: заказана апрель
- [ ] Договор с VPG FL-HYDRO подписан (июнь)

- [ ] Поиск инженера LMD начат (ноябрь)
- [ ] Выручка: 80–110 млн/год

Строительство:
- [ ] ГИ принят (Q1), операционка делегирована (Q2–Q3)
- [ ] Выход из оперативки (Q4)
- [ ] Выручка: 80 млн/год

Финансы:

- [ ] Доход: 250–280 тыс./мес
- [ ] Накопления: 50–70 тыс./мес

Личное:
- [ ] Вес: 88–90 кг
- [ ] Плавание: 1–2 км непрерывно
- [ ] Сын: переезд в Москву (лето)
