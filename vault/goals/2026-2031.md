# Цели по годам 2026–2031

---

## 2026 — Фундамент

**Главная тема: разгон и первые системные шаги**

### Бизнес
- [ ] Строительство: выручка 60–70 млн/год
- [ ] Строительство: аудит воронки клиентов + систематический поиск (застройщики, промпредприятия)
- [ ] Написать профиль «Главного инженера», открыть вакансию (Q3–Q4)
- [ ] Производство: первые устойчивые заказы, выход на безубыточность
- [ ] Производство: соглашение учредителей (письменно), P&L ежемесячно, точка безубыточности

### Финансы
- [ ] Доход: 180–200 тыс/мес
- [ ] Рассчитать остаток кредита, открыть счёт для досрочного погашения
- [ ] Погашено ~2,5–3 млн от кредита

### Личное
- [ ] Бассейн: 3 раза/нед к Q3
- [ ] Базовый медосмотр (ОАК, липиды, ЭКГ) — апрель
- [ ] Вес: −4–6 кг (→ 93–95 кг)
- [ ] Горный однодневный маршрут (Алтай / Кавказ / Крым)
- [ ] 2 отпуска: главный (море + город) + малый
- [ ] Определить направление вуза для сына, найти репетиторов

## 2027 — Рост и делегирование

**Главная тема: нанять главного инженера, закрыть долг**

### Бизнес
- [ ] Строительство: выручка 80 млн/год
- [ ] Главный инженер принят (Q1)
- [ ] Передача операционки ГИ (Q2–Q3): закупки, бригады, клиенты на объектах
- [ ] Выход из оперативки (Q4): остаётся стратегия + ключевые клиенты + финконтроль
- [ ] Производство: первые дивиденды, запуск EDM-станка, выход на прецизионные ниши, выручка 80–100 млн

### Финансы
- [ ] Доход: 250–280 тыс/мес
- [ ] Кредит полностью закрыт (Q1–Q2) — **Долг = 0**
- [ ] Старт инвестиционного счёта (Q3+): откладывать 50–70 тыс/мес

### Личное
- [ ] Вес: 88–90 кг
- [ ] Плавание: 1–2 км непрерывно
- [ ] Первый горный поход 1–2 дня (возможно с сыном)
- [ ] Сын: интенсивная подготовка, пробные ЕГЭ, поездка в Москву (дни открытых дверей)
- [ ] Лето: организация переезда сына в Москву

## 2028 — Масштаб и стабильность

**Главная тема: системный рост обоих бизнесов, здоровье достигнуто**

### Бизнес
- [ ] Строительство: выручка 100–120 млн, фокус на высокомаржинальные ниши (промышленное, спецобъекты)
- [ ] Производство: серийное производство в целевой нише, первые патенты/ноу-хау, выручка 150–200 млн

### Финансы
- [ ] Доход: 300–350 тыс/мес
- [ ] Инвестиционный капитал: +1,5–1,7 млн (накопления) + первые дивиденды реинвестированы
- [ ] Старт накоплений на квартиру сыну (параллельно)

### Личное
- [ ] **Целевой вес достигнут: 85–87 кг ✓**
- [ ] Плавание: 3–5 км в бассейне
- [ ] Горный поход средней сложности (4–5 дней: Карпаты / Грузия)
- [ ] Сын: 1–2 год в университете, самостоятельный

## 2029 — Три ноги стула

**Главная тема: все три источника дохода работают**

### Бизнес
- [ ] Строительство: выручка 150 млн
- [ ] Производство: выручка 200 млн, НИОКР как конкурентное преимущество

### Финансы
- [ ] Доход: 380–420 тыс/мес
- [ ] Инвестиционный капитал: 5–7 млн

### Личное
- [ ] Плавание: уверенно 3–5 км, рассматривать соревнования
- [ ] Горный поход: Непал на горизонте
- [ ] Квартира сыну: накопления идут по плану

## 2030–2031 — Цель достигнута

**Главная тема: все три источника, выход на крейсерскую скорость**

### Бизнес
- [ ] Общая выручка двух бизнесов: ~450 млн
- [ ] Роль: стратегия и развитие, не операционка

### Финансы
- [ ] **Доход: 450 000 руб/мес ✓**
  - 250 000 — строительство
  - 100 000 — дивиденды производства
  - 100 000 — проценты с капитала
- [ ] **Инвестиционный капитал: 14–15 млн ✓**
- [ ] Долгов: 0

### Личное
- [ ] **Плавание: 5 км без остановки ✓**
- [ ] **Вес: стабильно 85–87 кг ✓**
- [ ] Горный поход 4–5 дней (Непал / Грузия)
- [ ] Сын: студент, самостоятельный, перспектива квартиры есть
- [ ] Пересмотр стратегии на 2031–2036
